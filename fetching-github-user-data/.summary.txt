# Fetching GitHub User Data - Skill Summary

## Skill Created Successfully ✓

This skill allows users to fetch comprehensive GitHub user data through a simple command.

## What It Does

Fetches all available GitHub user data including:
- Profile information
- Repositories (with detailed statistics)
- Contributions calendar
- Pull requests and issues
- Programming language distribution
- Social connections
- Gists and starred repositories

## Quick Usage

# Basic (no token)
python scripts/fetch.py torvalds

# With token (recommended)
export GITHUB_TOKEN="ghp_YOUR_TOKEN"
python scripts/fetch.py torvalds --output ./my_data
